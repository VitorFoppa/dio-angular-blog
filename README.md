# Aula Web Blog Angular

Projeto desenvolvido com Angular inspirado no meu blog pessoal original, evoluindo uma aplicação HTML/CSS/JavaScript puro para uma arquitetura moderna baseada em componentes.

---

## Sobre o Projeto

Este projeto foi criado durante meus estudos de Angular com o objetivo de praticar:

- Componentização
- Data Binding
- Rotas
- Organização de aplicações Angular
- Dark Mode dinâmico
- Responsividade
- Estrutura moderna de frontend

Além disso, o projeto também funciona como meu blog/portfólio pessoal.

## Como Executar

Clone o projeto:

```bash
git clone https://github.com/VitorFoppa/aula-web-blog-angular.git
```

Entre na pasta:

```bash
cd aula-web-blog-angular
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
ng serve
```

Abra no navegador:

```bash
http://localhost:4200
```
